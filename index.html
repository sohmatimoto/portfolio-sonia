SohBooks: Recomendações de Livros

O SohBooks é um site criado para recomendar livros de diversos gêneros, disponibilizando sinopses e informações sobre os autores. Foi feito seguindo as instruções da plataforma Alura e utiliza HTML e CSS para criar a estrutura e o design do site, e foi usado o JavaScript para adicionar interatividade.

O design foi inspirado no Site AluraBooks, com cores suaves e aconchegantes. A intenção do site é despertar o desejo da leitura nos navegadores, gerando a curiosidade de mergulhar no mundo literário e explorar diversos gêneros.

A navegação no site é simples e os livros estão separados conforme o seu gênero, o que facilita o usuário a encontrar o livro que deseja de acordo com o seu interesse.

Esse projeto é uma maneira de exibir os livros, servindo como um catálogo, onde o usuário pode ver uma lista de livros, com informações como título, autor, imagem de capa e descrição. O site pode ser uma forma de exibir dados sobre livros de uma maneira organizada e fácil de navegar.

Minha Biblioteca: Uma Webpage Personalizada

Duo Brownie é uma plataforma digital criada para apresentar os produtos de uma confeitaria, fundada por duas amigas, de forma agradável e acessível. O site foi desenvolvido seguindo as orientações da plataforma Alura, utilizando HTML e CSS para construir sua estrutura e design visual.

Inspirado no site Marcelo, o design adota uma paleta de cores que mistura tons de roxo e lilás, com o objetivo de criar um visual suave e acolhedor. O intuito é proporcionar uma experiência visual que estimule o visitante a explorar o cardápio da confeitaria de maneira envolvente.

A navegação é simples e intuitiva, com os produtos organizados de forma clara e objetiva, facilitando a busca do usuário por aquilo que deseja. A proposta é apresentar a confeitaria de forma direta e charmosa, destacando os doces e tornando a navegação no site uma experiência agradável.

Este projeto é uma excelente maneira de exibir os produtos de uma confeitaria ou qualquer outro tipo de negócio, proporcionando uma apresentação simples, porém atraente, sem exageros, mas com um visual que chama a atenção.

Minha Biblioteca: Uma Webpage Personalizada

O site Flashcards foi desenvolvido no intuito de ajudar na memorização ou no aprendizado de algum conteúdo, de uma forma simples e conceitual. O processo de desempenhar o site foi seguindo as instruções da plataforma Alura, utilizando o HTML para estruturar a página e os flashcards, CSS para estilizar a página e os elementos visuais e o JavaScript para a interatividade, como a troca de cartões (flashcards) e a lógica de apresentação das perguntas e respostas.

O design foi mantido como no Site de Flashcards do Alura, alterando somente as perguntas. A funcionalidade principal desse site é a de flashcards, que é um método eficiente para revisar e aprender informações. Os flashcards permitem ao usuário interagir de maneira simples, virando o cartão para ver a resposta e navegando por diferentes perguntas e respostas.

O objetivo desse site é proporcionar uma maneira simples e eficaz de revisar informações ou estudar com o uso de flashcards. A interatividade é central para o site, permitindo que os usuários explorem questões e respostas de maneira dinâmica, sem complicações.
