Análise Exploratória de Dados sobre Hábitos de Saúde e Bem-Estar

Este projeto é uma análise exploratória de dados sobre hábitos de saúde e bem-estar coletados por meio de um questionário, feito em Python. O objetivo foi investigar a frequência de atividade física, o consumo de frutas e bebidas açucaradas, a relação entre atividade física e consumo saudável, e os padrões de sono e tempo de tela dos participantes.

A análise incluiu a visualização de dados através de histogramas e a obtenção de estatísticas descritivas como média e mediana para as diferentes variáveis.

Bondinho: Verificação de Capacidade

Este projeto em Python aborda um problema de verificação de capacidade de um bondinho.

O objetivo é determinar se um grupo de alunos e monitores pode ser transportado em uma única viagem, respeitando a capacidade máxima do bondinho.

O programa recebe como entrada o número de alunos e o número de monitores, e verifica se a soma é menor ou igual à capacidade máxima (50 pessoas no problema original, e 30 no desafio).

Este projeto demonstra o uso de estruturas condicionais (if/else) para tomar decisões com base em dados de entrada.
